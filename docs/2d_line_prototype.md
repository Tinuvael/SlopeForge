# Прототип 2D-плана Datamine

Это изолированный инструмент проверки геометрии и ручного выбора сегментов уступа. Он не использует PostgreSQL, Alembic, PostGIS или производственные сущности SlopeForge. Данные прототипа не считаются производственными.

## CSV и Dataset

Поддерживаются CSV Datamine с колонками `X`/`XP`, `Y`/`YP`, `Z`/`ZP`, `SID` и, при наличии, `PTN`, `PID`, `TYPE`, `PVALUE`. `SID` — идентификатор исходной линии, `PTN` — порядок точки. Разделитель определяется автоматически (запятая, `;`, tab) либо выбирается пользователем. Поддержаны `utf-8-sig` и `utf-8`.

Каждый импорт создаёт Dataset (`D-001`, `D-002` и т. д.). Сегмент хранит `dataset_id`, SID и извлечённую замороженную геометрию. Поэтому новый CSV не меняет уже созданный уступ. Одинаковый SID в разных Dataset — это разные версии линии.

## Граница карьера

Выберите линию обычным кликом и в карточке **Информация** нажмите **«Назначить границей карьера»**. Повторное нажатие снимает признак. Он хранится в `DatamineLine.semantic_role` как `normal` или `pit_boundary`; это не TYPE и не роль сегмента.

Граница карьера показана устойчивой тёмной линией толще обычного контура, но слабее выбранной линии и сегментов. Её можно скрыть флажком **«Показывать границу карьера»**. Такой контур можно использовать как верхнюю, нижнюю или промежуточную границу уступа: геометрический workflow для него такой же.

## Работа с уступом

1. Нажмите **«Создать сегмент уступа»**.
2. Выберите нижнюю линию, поставьте A и B, подтвердите сегмент.
3. Так же выберите верхний сегмент и создайте `U-001`.
4. При необходимости выберите `U-001` и вручную добавляйте промежуточные сегменты кнопкой **«Добавить промежуточный сегмент»**.
5. Нажмите **«Готово»**. Уступ получит статус `ready`, временные маркеры и рабочая подсветка исчезнут.

Во время работы статус черновика — `editing`; после **«Готово»** — `ready`. Кнопка **«Редактировать»** сохраняет временную копию сегментов. **«Отменить изменения»** возвращает исходную геометрию, а **«Готово»** подтверждает изменения.

Автоматических предложений промежуточных линий нет: все они добавляются только вручную.

## Занятые участки

Для активной линии прототип показывает красным интервалы, уже используемые сохранёнными сегментами того же Dataset и SID. Tooltip объясняет, например: «Участок используется в U-003 как верхняя граница».

Маркер A нельзя поставить внутри красного интервала. Маркер B нельзя поставить, если участок между A и B имеет положительное перекрытие с занятым интервалом. Простое касание общей конечной точкой допускается. При неверном выборе workflow показывает короткое сообщение без модального окна; красный preview заранее показывает конфликт. Финальная проверка при сохранении остаётся вторым уровнем защиты.

## Вид и клавиши

План занимает примерно 75% окна. Рабочий горизонт меняется без сдвига камеры через `Ctrl+колесо` или стрелки `↑` / `↓`; текущая отметка видна в углу плана. В нейтральном режиме `Escape` очищает выделения, горизонт и подсветку. В workflow `Escape` последовательно убирает незавершённые маркеры и возвращает к выбору линии, не заменяя кнопку **«Готово»**.

## Хранение

JSON сохраняется в `SlopeForge/prototype/line_segment_prototype.json` внутри `QStandardPaths.AppDataLocation`, с fallback `~/.config/SlopeForge/prototype/line_segment_prototype.json`. Старые JSON с полями автоматических candidates читаются безопасно и эти поля игнорируются.

## Ограничения

- Нет 3D-каркасов, PyVista, VTK, OpenGL и полноценного GIS.
- Нет расчёта RMS, перебора, недобора и связи с взрывными событиями.
- Нет автоматической перепривязки старого уступа к новому Dataset.
- Автосвязь концов и multi-line selection удалены из рабочего процесса.

## Редактирование и клавиши

`Enter` подтверждает следующий допустимый шаг, который определяет state machine: выбранную линию, сегмент, создание уступа, добавление промежуточного сегмента или сохранение редактирования. `Backspace` возвращает на шаг назад. `Delete` удаляет явно выбранный промежуточный сегмент или, после подтверждения, выбранный уступ. Когда курсор находится в текстовом поле, эти горячие клавиши не перехватываются.

После выбора уступа на плане показаны его верхняя, нижняя и все промежуточные границы. В карточке можно выбрать промежуточный сегмент, увидеть SID, Z и длину, изменить или удалить его. Для верхней и нижней границ есть отдельные кнопки изменения.

При редактировании исходная линия и оба маркера A/B становятся активными. Их можно перетаскивать только вдоль этой линии; preview и длина обновляются. До сохранения изменяется только временный черновик. **«Отменить изменения»** возвращает snapshot, **Enter** или **«Сохранить»** заменяет frozen geometry сегмента. Занятость самого редактируемого сегмента временно исключается из проверки.

Признак границы карьера отображается только для выбранной исходной линии в обычном режиме; он не является свойством уступа или сегмента.
